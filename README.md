jVoxelSpace
===================

VoxelSpace engine demo, based on [original code](https://github.com/s-macke/VoxelSpace) of [Sebastian Macke](https://github.com/s-macke).
Written in Java.

Requires
--------
gradle for build.
```
gradle build
```

Controls
--------
| Action        | Control                       |
|---------------|-------------------------------|
| Fly           | WASD, Arrows, or mouse click  |
| Pitch         | E/C                           |
| Fly vertical  | Q/Z                           |
| MouseLook     | M                             |
